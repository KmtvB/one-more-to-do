# One More ToDo
## About
[Link to heroku](http://one-more-to-do.herokuapp.com/)
## Features
* Pages as groups, pages can have title
* Deleting tasks
* The rest is the same as others app has
## Build&Run
1. `clone repo`
2. `npm install`
3. wait eternity
4. `npm start` 
5. go to [localhost:3000](http//localhost:3000/)
6. Enjoy! (or not).
